## 部署说明

### 使用GitBook官网
1. 访问 [GitBook.com](https://www.gitbook.com/)
2. 连接GitHub仓库
3. 选择这个仓库
4. 一键发布

### 使用GitHub Pages
1. 进入仓库Settings
2. 找到Pages选项
3. 选择Source为GitHub Actions
4. 等待自动部署完成
